📱 iOS Shortcut Setup

This Shortcut auto-saves every bank SMS to a JSON file in your Files app. Then open this app and tap 📂 to load that file.

1
Open Shortcuts app → Automation tab → ＋ → Create Personal Automation
2
Choose Message → Sender Contains → add your bank IDs:
HDFCBK, ICICIB, SBIBNK, AXISBK, KOTAKB, PNBSMS, YESBK
3
Add these actions in order (see code block below)
4
Turn OFF "Ask Before Running" → Done!
5
Open this app anytime → tap 📂 → pick bank_sms.json from Files

Shortcut Actions:

Action 1: Text ───────────────────────── {"message":"[Shortcut Input]","sender":"[Sender]","timestamp":"[Current Date]"} Action 2: Get File ───────────────────────── File: Shortcuts/bank_sms.json (If Not Found → skip to Action 5) Action 3: Get Text from [File] ───────────────────────── Action 4: Text ───────────────────────── (Check if file was empty or had content) If old text starts with {"messages": → Replace closing ]} with: ,{new entry}]} Else → go to Action 5 Action 5 (first time / fallback): Text ───────────────────────── {"messages":[{entry from Action 1}]} Action 6: Save File ───────────────────────── Save to: Shortcuts/bank_sms.json (Ask Where: OFF, Overwrite: ON)

Simpler alternative: Use a Shortcut that just appends each SMS as a new line to a .txt file. This app can also parse plain-text files with one SMS per line.

Simple version (append text): ───────────────────────── Action 1: Append Text Text: [Shortcut Input] File: Shortcuts/bank_sms.txt Make New Line: ON Then load the .txt file in this app → 📂

Tip: The app auto-deduplicates, so loading the same file twice won't create duplicates.
