Install Bank SMS Shortcut

Auto-save every bank transaction SMS to a file.
Then load it into Expense Tracker — that's it.

💬 Bank SMS → ⚡ Shortcut → 📄 File → 📊 App
Recommended Simple Text File
Appends each bank SMS as a new line to bank_sms.txt. Simplest setup — works perfectly with the app's smart parser.
Advanced JSON Format
Builds a structured JSON file with message, sender & timestamp. Better metadata but same end result in the app.
After downloading
⚡ Complete Setup (2 min)
1
Open the downloaded file — iOS will ask to add it to the Shortcuts app. Tap Add Shortcut.
2
Open Shortcuts app → Automation tab → tap ＋ → Create Personal Automation
3
Choose Message trigger → Sender Contains → add your bank IDs:
HDFCBK ICICIB SBIBNK AXISBK KOTAKB PNBSMS YESBK INDUSBK FEDBNK IDFCFB CANBNK BOBTXN
Tap any ID to copy it. Add only the banks you use.
4
Set the action to Run Shortcut → choose "Save Bank SMS" (the one you just installed)
5
Turn OFF "Ask Before Running" → tap Done
All done! Every bank SMS will auto-save to iCloud Drive → Shortcuts → bank_sms file. Open the Expense Tracker app → tap 📂 → select the file to import.
How import works: Open the app → tap 📂 → pick bank_sms.txt (or .json) from Files. The app parses all SMS, deduplicates, and adds new transactions. You can import the same file repeatedly — duplicates are auto-skipped.
Install the App
🏠 Add to Home Screen (Safari)
Install Expense Tracker as a full-screen app on your iPhone or iPad. It works just like a native app — with its own icon, splash screen, and no browser UI.
1
Open Safari on your iPhone/iPad and navigate to the Expense Tracker URL.
⚠️ This only works in Safari — Chrome/Firefox on iOS don't support Add to Home Screen.
2
Tap the Share button ⬆️ at the bottom of Safari (the square with an arrow pointing up).
3
Scroll down in the share sheet and tap "Add to Home Screen" ➕.
4
You'll see a preview with the app name and icon. Tap Add in the top-right corner.
5
The app icon now appears on your Home Screen. Tap it to launch — it opens full-screen without any browser bars.
Why Safari? Apple only allows PWA installation through Safari. When installed, the app gets its own entry in the App Switcher, can receive push notifications (iOS 16.4+), and stores data independently from Safari.
Using Offline
📡 How Offline Mode Works
Expense Tracker is a fully offline-capable PWA. Once loaded, it works without any internet connection — no server, no cloud, no accounts.
1
First visit caches everything — when you open the app for the first time, the Service Worker downloads all files (HTML, CSS, JS, icons) and stores them locally on your device.
2
All data stays on your device — transactions are saved in your browser's IndexedDB. Nothing is ever sent to a server. Your financial data never leaves your phone.
3
Works in airplane mode — open the app from your Home Screen anytime. Browse transactions, view charts, search, filter — everything works without WiFi or cellular data.
4
Import files offline too — the SMS parser runs entirely in JavaScript on your device. You can import bank_sms.txt even without internet. Just open Files → pick the file → done.
5
Export anytime — tap the export button to download your data as CSV or JSON. Works offline. Use it for spreadsheets, budgeting apps, or backup.
Tip: After installing to Home Screen, open the app once while online to ensure all assets are cached. After that, it's fully offline forever — until you manually clear Safari website data.

🛠️ Troubleshooting
?
"Add to Home Screen" not showing? — Make sure you're using Safari, not Chrome or an in-app browser. Also check Settings → Safari → Advanced → ensure "Web Inspector" isn't blocking PWAs.
?
Shortcut not triggering? — Go to Shortcuts → Automation → check your SMS automation is enabled and "Ask Before Running" is OFF. Also verify the sender ID matches exactly (e.g., HDFCBK not HDFC).
?
File not found when importing? — The shortcut saves to iCloud Drive → Shortcuts folder. In the Files app, browse to "iCloud Drive → Shortcuts → bank_sms.txt". Make sure iCloud Drive is enabled in Settings.
?
App not working offline? — Open it once while connected to the internet so the Service Worker can cache all files. Then it works offline. If still broken, delete the Home Screen icon and re-add it from Safari.
?
Lost my data? — Data lives in IndexedDB. Clearing Safari website data or resetting the phone will erase it. Use the export feature regularly to back up to CSV/JSON files.
