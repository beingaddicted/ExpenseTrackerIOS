Setup Guide

Export bank SMS from your iPhone → import into Expense Tracker.
All offline, all private, all on your device.

Step 1 — Bulk Export Past SMS
Start Here Export All Historical SMS
Export all your past bank SMS in one go using Scriptable (free app) + an iOS Shortcut.
1
Install Scriptable from the App Store (free).
2
Download the script, then copy-paste it into Scriptable:
Open Scriptable → tap ＋ → paste the script → rename to BankSMS → Done.
3
Install the Shortcut that drives the export:
⬇️ Install Bulk Export Shortcut
4
Open Shortcuts app → tap the shortcut to run. It processes every day from 2020 to today.
5
Output file: Files → iCloud Drive → Scriptable → expense tracker → exportSms.txt
Crash safe: If the shortcut stops mid-run, just re-run — it resumes automatically. No duplicates.
Step 2 — Import into the App
Quick Load Transactions
Open Expense Tracker → tap 📂 Import → pick exportSms.txt (or bank_sms.txt for real-time captures). The app parses amount, merchant, category, date, and bank automatically. Duplicates are always skipped.
Import as often as you like — dedup by amount + date + merchant + reference ensures no double entries.
Step 3 — Auto-Save New SMS (Optional)
Bulk export gets your history. This optional step captures every future bank SMS automatically to a file, so you just import periodically.
1
Shortcuts app → ＋ → Add Action → search "Append" → tap "Append to Text File".
2
Configure the action:
  • Text field → tap "Shortcut Input"
  • File path → type bank_sms.txt
  • Append on New Line → ON
3
Rename the shortcut to Save Bank SMS → Done.
Create one automation per keyword. iOS only allows one "Message Contains" value per automation.
1
Shortcuts → Automation tab → ＋ → Message.
2
Leave Sender empty. Set "Message Contains" → debit.
3
Next → Add Action → search "Run Shortcut" → pick "Save Bank SMS".
4
Turn OFF "Ask Before Running" → Done.
Repeat for these keywords (one automation each):
debit credit sent spent
Tap any keyword to copy. These 4 cover 99% of Indian bank SMS.
Install the App
1
Open the Expense Tracker URL in Safari (not Chrome/Firefox).
2
Tap Share ⬆️ → "Add to Home Screen" → Add.
3
Launch from Home Screen — full-screen, no browser bars, works offline after first load.
Troubleshooting
?
Automation not triggering? — Ensure "Ask Before Running" is OFF. Check the automation toggle is enabled. Try disabling Focus/DND.
?
File not found when importing? — Bulk export saves to iCloud Drive → Scriptable → expense tracker. Real-time saves to iCloud Drive → Shortcuts. Make sure iCloud Drive is enabled.
?
"Add to Home Screen" missing? — Use Safari only. Chrome/Firefox on iOS don't support PWA install.
?
App not working offline? — Open it once while online to cache all files. If still broken, remove and re-add from Safari.
