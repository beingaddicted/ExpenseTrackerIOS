Or Build It Manually
🔧 Manual 9-Action Shortcut
If you prefer to build the bulk export shortcut yourself instead of using the install link above, follow the step-by-step guide to create all 9 actions manually.
Real-Time — Auto-Save New SMS
⬇️ Bulk export gets your past SMS. The sections below set up real-time capture so every new bank SMS is saved automatically going forward.
Part 1 — Create the Shortcut
Step A Create "Save Bank SMS" Shortcut
This shortcut saves the incoming SMS text to a file. You'll build it once — it takes about 1 minute.
1
Open the Shortcuts app on your iPhone. If you don't have it, download it free from the App Store.
2
Tap the Shortcuts tab at the bottom (not Automation), then tap the ＋ button in the top-right corner to create a new shortcut.
3
You'll see the shortcut editor. Tap "Add Action" — a search sheet appears.
4
In the search bar at the top, type "Append". You'll see "Append to Text File" in the results. Tap it to add it.
Can't find it? Try searching "Append to File" or just "Text File". On some iOS versions it may appear as "Append to File" under Documents.
5
Now configure the action. You'll see it has blue/grey tappable fields:
  • Tap the "Text" field (where it says what to append). Tap "Shortcut Input" from the variable suggestions above the keyboard. This passes the SMS message body into the shortcut.
  • Tap the file path field. Type exactly: bank_sms.txt
    The file will be created automatically in iCloud Drive → Shortcuts folder.
  • If you see a "Make New Line" or "Append on New Line" toggle — make sure it's ON (it usually is by default).
6
Tap the shortcut name at the very top (it might say "New Shortcut" or show a dropdown arrow). Rename it to:
Save Bank SMS
7
Tap Done (top-right). Your shortcut is saved. You can see it in the Shortcuts tab.
Test it: Go back to the Shortcuts tab, tap "Save Bank SMS", type any text when prompted, then check Files app → iCloud Drive → Shortcuts — you should see bank_sms.txt with your text.
Part 2 — Set Up SMS Automation
Step B Auto-Trigger on Bank SMS
Now you'll tell iOS to run your shortcut automatically every time a bank SMS arrives. This is the key step.
1
In the Shortcuts app, tap the Automation tab at the bottom.
2
Tap the ＋ button in the top-right corner (or if this is your first automation, you may see a "Create Personal Automation" button directly).
3
Tap "Create Personal Automation" if you see the option. Then scroll down and tap "Message" under the Communication section.
4
Configure the Message trigger — use "Message Contains" instead of Sender (iOS strips hyphens from sender IDs, making them unreliable):
  • Leave "Sender" empty.
  • Tap "Message Contains" → type debit
This catches all debit SMS (UPI, NEFT, IMPS, card swipes). The keyword "debit" matches "debited", "debit card", "debit of Rs." etc.
5
Tap Next.
6
You'll see options for what action to perform. Tap "New Blank Automation" or "Add Action".
7
In the action search, type "Run" and select "Run Shortcut".
8
Tap the blue "Shortcut" placeholder text → select "Save Bank SMS" (the one you created in Part 1).
9
Tap Next. On the summary screen:
  • Turn OFF the "Ask Before Running" toggle — this lets it run silently in the background without a popup every time.
  • If you see "Notify When Run", you can turn it off too (optional — keeping it on gives you a small confirmation banner).
10
Tap Done. Your first automation is set.
Now create 3 more automations!
Repeat the exact same steps above for each of these keywords (one automation per keyword):
credit sent spent
• debit → catches "debited", "debit card", all outgoing transactions
• credit → catches "credited", "credit card", salary, refunds
• sent → catches "sent", UPI transfers with "sent to" phrasing
• spent → catches "spent on", "spent at", card purchase alerts
Why 4 separate automations? iOS only allows one "Message Contains" value per automation. These 4 keywords together cover 99% of Indian bank transaction SMS while ignoring OTPs, balance checks, promos, and other junk. The app's parser handles any duplicates or non-transaction messages automatically.
Want even more coverage? Optionally create automations for:
withdrawn purchased received transferred payment
Tap any keyword to copy. The 4 main keywords (debit + credit + sent + spent) cover most cases.
Part 3 — Import into the App
Step C Import SMS into Expense Tracker
Once you have some bank SMS collected, import them into the app. Do this whenever you want — weekly, monthly, or anytime.
1
Open Expense Tracker from your Home Screen (or Safari).
2
Tap the 📂 Import button.
3
The iOS file picker opens. Navigate to: iCloud Drive → Shortcuts → bank_sms.txt
Don't see the Shortcuts folder? Tap "Browse" at the bottom of the Files picker → iCloud Drive. The Shortcuts folder is created automatically when the shortcut first saves a file.
4
Select the file. The app parses every SMS — extracting amount, merchant, category, date, bank — and adds them as transactions. Duplicates are automatically skipped.
You can import the same file repeatedly — the app deduplicates by matching amount + date + merchant + reference number. No double entries ever.
Install the App
🏠 Add to Home Screen (Safari)
Install Expense Tracker as a full-screen app on your iPhone or iPad. It works just like a native app — with its own icon, splash screen, and no browser UI.
1
Open Safari on your iPhone/iPad and navigate to the Expense Tracker URL.
⚠️ This only works in Safari — Chrome/Firefox on iOS don't support Add to Home Screen.
2
Tap the Share button ⬆️ at the bottom of Safari (the square with an arrow pointing up).
3
Scroll down in the share sheet and tap "Add to Home Screen" ➕.
4
You'll see a preview with the app name and icon. Tap Add in the top-right corner.
5
The app icon now appears on your Home Screen. Tap it to launch — it opens full-screen without any browser bars.
Why Safari? Apple only allows PWA installation through Safari. When installed, the app gets its own entry in the App Switcher, can receive push notifications (iOS 16.4+), and stores data independently from Safari.
Using Offline
📡 How Offline Mode Works
Expense Tracker is a fully offline-capable PWA. Once loaded, it works without any internet connection — no server, no cloud, no accounts.
1
First visit caches everything — when you open the app for the first time, the Service Worker downloads all files (HTML, CSS, JS, icons) and stores them locally on your device.
2
All data stays on your device — transactions are saved in your browser's IndexedDB. Nothing is ever sent to a server. Your financial data never leaves your phone.
3
Works in airplane mode — open the app from your Home Screen anytime. Browse transactions, view charts, search, filter — everything works without WiFi or cellular data.
4
Import files offline too — the SMS parser runs entirely in JavaScript on your device. You can import bank_sms.txt even without internet. Just open Files → pick the file → done.
5
Export anytime — tap the export button to download your data as CSV or JSON. Works offline. Use it for spreadsheets, budgeting apps, or backup.
Tip: After installing to Home Screen, open the app once while online to ensure all assets are cached. After that, it's fully offline forever — until you manually clear Safari website data.

🛠️ Troubleshooting
?
"Add to Home Screen" not showing? — Make sure you're using Safari, not Chrome or an in-app browser. Also check Settings → Safari → Advanced → ensure "Web Inspector" isn't blocking PWAs.
?
Automation not triggering? Check these in order:
  • Keyword issue: Make sure your automation uses "Message Contains" with exactly debited or credited. Don't use "Sender" — iOS strips hyphens from sender IDs.
  • "Ask Before Running" is ON: Go to Shortcuts → Automation → tap your automation → make sure "Ask Before Running" is OFF.
  • Automation is disabled: In the Automation tab, check the toggle next to your automation is green/enabled.
  • Focus mode blocking: If you have Focus/DND active, it may suppress automations. Try disabling Focus temporarily.
  • Test manually first: Open Shortcuts → tap "Save Bank SMS" → type any text → confirm a file appears in Files → iCloud Drive → Shortcuts.
?
File not found when importing? — The shortcut saves to iCloud Drive → Shortcuts folder. In the Files app, browse to "iCloud Drive → Shortcuts → bank_sms.txt". Make sure iCloud Drive is enabled in Settings.
?
App not working offline? — Open it once while connected to the internet so the Service Worker can cache all files. Then it works offline. If still broken, delete the Home Screen icon and re-add it from Safari.
?
Lost my data? — Data lives in IndexedDB. Clearing Safari website data or resetting the phone will erase it. Use the export feature regularly to back up to CSV/JSON files.
